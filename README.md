## How to run file
### Just run it in cmd <br>

The first thing you need is to install **raylib** library: <br>
```
brew install raylib
```
After go to the Maze/build directory and sequentially write it: <br>
```
mkdir build 
```
```
cd build 
```
``` 
cmake .. 
```
```
cmake --build . 
```
And finaly this: <br>
```
./Maze
```
Enjoy the game!!!

**ADDITIONALLY** add --help to find more about the game
```
./Maze --help
```

## Controlls
* W: Move up
* A: Move left
* S: Move down
* D: Move right
