## How to run file
### Just run it in cmd <br>

The first thing you need is to install **raylib** library: <br>
```
brew install raylib
```
After just go to the Maze/build directory: <br>
```
cd path/to/Maze/build
```
And run this: <br>
```
./Maze
```
Enjoy the game!!!

**ADDITIONALLY** add --help to find more about the game

## Controlls
* W: Move up
* A: Move left
* S: Move down
* D: Move right
